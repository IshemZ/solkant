# Images OpenGraph

Ce dossier contient les images OpenGraph (og:image) pour le partage social.

## Spécifications recommandées

- **Dimensions** : 1200 x 630 pixels
- **Format** : PNG ou JPEG (WebP supporté)
- **Poids** : < 1 MB
- **Ratio** : 1.91:1

## Images requises

1. `home.png` - Homepage (Solkant logo + slogan)
2. `pricing.png` - Page tarifs
3. `features.png` - Page fonctionnalités
4. `contact.png` - Page contact
5. `blog.png` - Page blog
6. `article-devis.png` - Article "Comment faire un devis"
7. `article-logiciel.png` - Article "Choisir logiciel"

## Génération temporaire

En attendant les vraies images design, nous utilisons des placeholders depuis https://placehold.co/

Pour créer de vraies images :

- Utiliser Canva (template "Facebook Post" 1200x630)
- Inclure : logo Solkant + titre de page + fond élégant
- Exporter en PNG haute qualité
